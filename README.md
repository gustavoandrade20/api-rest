# 📚 API REST com Node.js, Express e MySQL

Este projeto é uma **API REST completa** desenvolvida com **Node.js**, **Express**, **MySQL** e **JavaScript (ES6+)**.  
Foi criada com foco em boas práticas, segurança e organização seguindo a arquitetura **MVC**.

---

## 🚀 Tecnologias Utilizadas

- **Node.js**
- **Express**
- **MySQL (mysql2)**
- **Sequelize (ORM)**
- **Multer (upload de imagens)**
- **JWT (autenticação)**
- **Insomnia (testes da API)**

---

## 📌 Funcionalidades da API

### 👤 Usuários
- Criar usuário  
- Fazer login com JWT  
- Listar usuários  
- Atualizar usuário  
- Deletar usuário  

### 🧑‍🎓 Alunos
- CRUD completo (Create, Read, Update, Delete)  
- Validações de nome, email, idade, altura e peso  

### 📷 Upload de Fotos
- Upload usando Multer  
- Associação da foto com um aluno  
- Geração automática de URL
